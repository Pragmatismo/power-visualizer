# Requirements

## Runtime requirements

- **Python**: 3.9+ recommended.
- **Operating system**: Windows, macOS, or Linux with a graphical desktop environment.
- **Dependencies**:
  - [PySide6](https://pypi.org/project/PySide6/) (Qt-based GUI toolkit)
  - [astral](https://pypi.org/project/astral/) (sunrise/sunset calculations)

## Optional assets

- Device catalog JSON files, such as [basic-devices.json](basic-devices.json).
- Tariff schedule JSON files following the `uk-tariffs-1.0` schema, such as [UK-tariffs.json](UK-tariffs.json).
